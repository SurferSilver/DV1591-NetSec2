# Network Security Course

This is a personal repo for the Network Security 2 course at BTH.

## Repository Structure

Each branch in this repository corresponds to a different part of the course:

| Branch | Description |
|--------|-------------|
| `main` | Main branch with general course information |
| `Threat-Intelligence` | Project on threat actor usage of AI tools |

## Switching Between Projects

To view a specific course project, switch to the corresponding branch:

```bash
git checkout <branch-name>
```

For example:
```bash
git checkout Threat-Intelligence
```
